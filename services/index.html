Rust Language Services by Ivan Dardi

Since I started learning Rust around 8 years ago, I made it a goal of mine to have it as the center of my career. It took a while, but I'm now currently working with Rust in the scope of modern mobility solutions. I'm very happy with my current job, and it gives me confidence to also offer services related to Rust. Here are some of the services I'm offering:

1-on-1 Rust Tutoring

I offer 1-on-1 Rust tutoring sessions. These sessions are for everyone, whether you're a beginner or an experienced Rustacean. I can help you with, but not limited to:

  • Learning the basics of Rust
  • Understanding Rust's ownership system
  • Writing idiomatic Rust code
  • Debugging Rust code
  • Writing projects in Rust

If you're interested in those sessions, please contact me.

Talks and Presentations

and doing talks and presentations is something I'm very interested in doing ever since I started going to Rust meetups and watching the RustConf talks.

I can talk about:

  • My journey with Rust and the community Discord server I created
  • The basics of Rust
  • Rust's ownership system
  • Writing idiomatic Rust code
  • Approaches on how to learn Rust in a way that works for you
  • Specific crates, libraries or tools written in Rust

If you'd like to invite me for a talk or presentation, please contact me.

Workshops

I really enjoy workshops where you can actually get a working project done by the end of it. I can host workshops about:

  • The basics of Rust
  • Writing an API with Rust and a popular web framework
  • Writing a CLI tool with Rust
  • Writing a GUI application with Rust
  • Writing a game with Rust
  • Writing a library or crate in Rust
  • Writing and using an FFI library

If you're interested in having me host a one-time-only workshop, or a series of workshops, please contact me.

Consulting

I can act as a consultant for your Rust project.

If you're interested in hiring me as a consultant, please contact me.
