I'm happy to announce the launch of my website "How did we get here?". This website is a place where I can share my thoughts, projects, and experiences with the world. I've been wanting to create a personal website for a long time, and I'm excited to finally have it up and running.

About the name

The name "How did we get here?" is a bit of a personal thing to me. It's not something I ask myself, but rather it's an observation of the world around me. The world feels very large and complex nowadays, and it's easy to get lost in it. If you look at me, you can say that I'm a software engineer that likes the Rust programming language. But that's not the whole story, it never is. 9 years ago, I've never even heard of Rust. I was a college student that was just starting to learn how to code data structures with C and C++. And now, I have a lot more knowledge not only about Rust but about... everything! And yet, there's still a lot more to learn. It's that old saying, "The more you know, the more you realize you don't know." That feels very true to me, and that's why I chose this name. To remind myself how I got here, and to make sure I keep enjoying the journey as it goes on. After all, it's hard to know where I'll end up next!

About the website

I built this website using Zola, a static site generator written in Rust. I chose Zola because I wanted to use Rust for my website, and Zola seemed good. I was debating on using Hugo, but I felt it'd be more thematic to use a Rust tool. Not only that, but the theme I'm currently using (Tabi) is one that I approve a lot. It has a responsive design, it's easy to work with, looks good, and it has great accessibility. I enjoy using tools that care about all those details that are often forgotten.

About services offered

This is something that I've been mulling on for a while. I already do a lot of helping people with Rust with my Discord server, but I've never been compensated for it. And every once in a while someone comes up there and asks if there are any tutors or teachers that can help them 1-on-1. So I thought "hey, I could do that!". Of course, there is always the nervousness that comes with offering services like that for money. It took me having a job in Rust to feel confident enough to start offering services related to Rust, but the nervousness will always be there. Regardless of it, I'd still like to give it a try, and hopefully help people in ways that I couldn't before.
