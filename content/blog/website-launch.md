I'm happy to announce the launch of my personal programming website "Programming Insights". This website is a place where I can share my
thoughts, projects, and experiences I've had with the programming world. I've been wanting to create a personal website for a long time, and
I'm excited to finally have it up and running.

# About the name

There are 2 hard problems when it comes to programming: cache invalidation, naming things, and off-by-1 errors.

I've had a hard time trying to come up with a good name for this website. I don't want something that has my name in it,
but I also want to something that indicates it's a personal blog dedicated to programming. For now I've settled with
"Programming Insights". Over the course of 8+ years of programming, I've accumulated a lot of insights about it, and
that's where the name comes from. It may change eventually as I continue to learn, evolve and get new ideas that are
hopefully better.

# About the website

I built this website using [Zola](https://www.getzola.org/), a static site generator written in Rust. I chose Zola
because I wanted to use Rust for my website, and Zola seemed good. I was debating on using [Hugo](https://gohugo.io/),
but I felt it'd be more thematic to use a Rust tool. Not only that, but the theme I'm currently using (Tabi) is one
that I approve a lot. It has a responsive design, it's easy to work with, looks good, and it has great accessibility.
I enjoy using tools that care about all those details that are often forgotten.

# About services offered

This is something that I've been mulling on for a while. I already do a lot of helping people with Rust with my Discord
server, but I've never been compensated for it. And every once in a while someone comes up there and asks if there are
any tutors or teachers that can help them 1-on-1. So I thought "hey, I could do that!". Of course, there is always the
nervousness that comes with offering services like that for money. It took me having a job in Rust to feel confident
enough to start offering services related to Rust, but the nervousness will always be there. Regardless of it, I'd still
like to give it a try, and hopefully help people in ways that I couldn't before.
